# **App Name**: DevFlow AI

## Core Features:

- AI Summary Tool: Summarize Git commits, PRs, and stand-up notes using Gemini. The LLM uses a tool to determine when a longer piece of text needs summarizing
- Meeting Parser: Parse meeting transcripts, extract action items, and push to Trello or Jira.
- Smart Scheduler: Integrate with Google Calendar to automate stand-ups, sprints, or follow-ups.
- Code Snippet Generator: Generate code snippets from natural language requests.
- Notification System: Send automation and approval notifications via Firebase Cloud Messaging.
- User Profile Storage: Store user profiles, automation templates, logs, and history in Firestore.
- Secure Authentication: Secure developer login using Google Sign-In via Firebase Auth.

## Style Guidelines:

- Primary color: Deep violet (#7955F4) to convey sophistication and innovation in the developer tools space.
- Background color: Light gray (#F5F5F5), a muted backdrop that maintains a clean, modern aesthetic, allows the primary color to pop, and ensures legibility.
- Accent color: Electric purple (#E040FB), a vivid counterpart that brings high-contrast dynamism to calls to action, interactive elements, and visual highlights.
- Headline font: 'Space Grotesk' (sans-serif) for a tech-forward, readable style in headers.
- Body font: 'Inter' (sans-serif) for a modern, neutral style in main content and descriptions.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist icons representing automation tasks and integrations.
- Subtle animations for task completion and loading states to enhance user experience.
